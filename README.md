# SAILSAFE

Autonomous surface catamaran focused on control systems development.

## Overview
SAILSAFE is a personal engineering project focused on building an autonomous surface vehicle with differential propulsion, embedded control, and future waypoint navigation.

## Current Phase
Phase 1 — architecture validation.

## Current Status
- System architecture defined and documented
- Raspberry Pi 4 initial setup completed
- Initial ESP32 firmware prepared
- Electrical schematic created
- Awaiting component integration and first bench tests

## Main Components
- Raspberry Pi 4
- ESP32
- 2 brushless motors with ESCs
- GPS
- IMU
- ADC

## Control Focus
The project is being developed as a real-world platform for learning and applying control systems, starting with:
- Differential thrust control
- Embedded motor control and safety logic
- Future heading and waypoint control

## Repository Structure
- `docs/` → architecture and project documentation
- `hardware/` → electrical and mechanical files
- `software/` → firmware and future onboard code

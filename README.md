# SAILSAFE

Autonomous surface catamaran focused on control systems development.

## Overview
SAILSAFE is a personal engineering project focused on building an autonomous surface vehicle with differential propulsion, embedded control, and future waypoint navigation.

## Current Status
- System architecture defined
- Raspberry Pi initial setup completed
- ESP32 firmware started
- Hardware integration in progress

## Main Components
- Raspberry Pi 4
- ESP32
- 2 brushless motors with ESCs
- GPS
- IMU
- ADC

## Repository Structure
- `docs/` → architecture and project documentation
- `hardware/` → electrical and mechanical files
- `software/` → firmware and future onboard code
- `media/` → photos and videos

## Focus
This project is being developed as a real-world platform for learning and applying control systems.
